o código se baseia sobre função do python, as funções em si envolvem calculos, ou seja, ela pede para o usuario números, e retorna o valor do calculo.

  Sendo 1 soma

  número 3 multiplicação
  número 4 divisão
  e número 5 raiz quadrada.
